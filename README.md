## 💻 Tech Stack

### Frontend Architecture
- **Framework**: React (Bootstrapped with Vite)
- **Styling Engine**: Custom CSS with Glassmorphism properties & dynamic animations
- **State Management**: React Context API
- **Icons**: Lucide React
- **Hosting**: Vercel

### Backend Engine
- **Server Environment**: Node.js with Express.js
- **Database**: MongoDB (Mongoose ODMs hooked to MongoDB Atlas)
- **Network**: CORS enabled, RESTful JSON architecture
- **Hosting**: Render.com

### AI & Deep Processing Pipeline
- **Generative Engine**: Google Gemini 2.5 Flash (Handles deep cognitive insights, quizzes, and formatting)
- **Audio AI**: Deepgram API (Nova-2 Model for ultra-fast, high-accuracy structural speech-to-text)
- **Media Extraction**: Custom autonomous wrapper for `yt-dlp` (Extracts native YouTube DASH `.m4a` streams on the fly without heavy FFmpeg transcoding requirements).
